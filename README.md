# PromoOps frontend assessment

This repository is the starting point for a 120-minute frontend coding
assessment. Read [ASSESSMENT.md](./ASSESSMENT.md) before making changes.

## Quick start

```bash
npm install
npm run dev
```

Open the URL printed by Vite. The development server defaults to port `5174`
and the application route is `/promotions`.

Run the automated tests and production build with:

```bash
npm test
npm run build
```

The separate high-volume catalogue scenario can be started with:

```bash
npm run dev:performance
```

The project intentionally contains an intermittent production defect and an
unfinished component. Work within the existing React, TypeScript, CSS, Vitest,
and React Router setup.
